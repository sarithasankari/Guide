# Guide Details Feature Layer

This feature manages the guide profile, credentials timeline, gallery slideshow lightbox, dynamic traveler reviews breakdown, and date calendar selections.

## Structure
- `components/` - Visual blocks (GuideHero, Gallery, BookingCard, FAQ, Reviews, etc.)
- `hooks/` - Custom react hooks (useGuide, useReviews)
- `services/` - Async endpoints mock services (guideService)
- `utils/` - Formatter helpers
- `constants/` - Static configurations
